# digital-lib — Shoot @ Sight Client Experience Portal

A high-fidelity digital client experience portal and photographer dashboard designed for **Shoot @ Sight Weddings**.

## Features

- **Private Access Screen**: Dynamic client authentication using secure event codes.
- **Client Memories Gallery**: Categorized photo grid with single download, multi-select download, ZIP compression builder, and gallery sharing features.
- **AI Face Finder**: Simulates a neural facial node scanner, matches a user's uploaded selfie against gallery photos, and filters matched frames.
- **Photographer Admin Dashboard**: Health metrics panel, new event creator, automatic access key generator, interactive image tag uploader, and active client logging.

## Tech Stack

- **Frontend**: Vanilla HTML5, CSS3, ES6 JavaScript modules.
- **Layout & Aesthetic**: Light brass premium theme utilizing Google Fonts (*Montserrat* and *Cormorant Garamond*).
- **Tooling**: Built to run out-of-the-box or via `vite`.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
